## 共通のボキャブラリ

# ボキャブラリはできるだけ、意味が特定されたものを選ぶ

増やす = 増やす|多くする|増加させる
減らす = 減らす|少なくする|減少させる
設定する = 設定する|セットする
カウントする=数える|カウントする

丸める = 四捨五入する|丸める
丸めた = 四捨五入した|丸めた

合計 = 合計値|総和|合計
平均 = 平均値|平均

接頭辞=始まり|始端|先頭
接尾辞=終わり|終端|末尾
始まる=始まる|開始する
終わる=終わる|終了する
先頭 = 最初|先頭
末尾 = 最後|末尾

見つかる = 見つかる|発見される|出現する
出現する=出現する|現れる|見つかる|発見される

インデックス=位置|インデックス
毎=毎|ごと

置換する=置き換える|置換する
置き換える=置き換える|置換する


## データ構造

整数=整数@2|自然数|整数値

表記=表記|表現|文字列
コードポイント=文字コード|コードポイント
エンコーディング=文字コード|エンコーディング
英大文字=大文字
英小文字=小文字

配列=配列|リスト
タプル=組|タプル
スライス = 部分列|スライス
番目=番目|

辞書=辞書@8|ハッシュ表|マップ
キー = 項目名|キー
エントリ = 項目|エントリ|値
アップデートする = 更新する|アップデートする

存在する = 存在する|ある@2
存在しない = 存在しない|ない@2
内 = 内@2|中@2|
